Introduction to SVG

SVG stands for scalable vector graphics and is somewhat similar to the HTML canvas element in that it is used to draw 2D graphics, although, while canvas creates a box to display JavaScript code pixel by pixel and is forgotten by the browser, SVG is part of the DOM and is remembered as an object. SVG images are created using XML and is best suited for large rendering areas. While SVG may not be coded using JavaScript, it does support the JS event handlers thus allowing a higher level of functionality.

Attributes

  • Coordinates

    • Uses X, and Y, to specify location

  • Width

    • Indicates the pidth of the object

  • Height

    • Indicates the height of the object

  • Fill

    • Fills the color inside the shape being drawn

  • Stroke

    • This is the border color for the shape.

  • Stroke-Width

    • This is the width of the border of the shape
