Making table data accessible

Certain elements of the table tag can be included in your code to make the table items to more accessible to screen readers and web browsers. <caption> can be placed as a first child of the table tag to give tables a name. <th> table headers specify the data types of each column in a table and should be used with the scope="attribute to specify exactly which data cells are associated with each column". A table attribute summary="gives a brief description of the table data", although this was removed from HTML5, there is no clear alternative to it's use and older versions of HTML and most screen readers still recognize this attribute.

Controlling Styling

Understanding the relationship between HTML and CSS

When HTML is displayed in a browser without user defined CSS styles the content will displayed user the default styles provided by the browser. HTML primarily acts as a structural language to organize content, CSS on the other hand works alongside CSS by influencing the presentation. CSS is coded by first declaring a selector which determines which HTML element to be effected, followed by a propery which determines which style element will be used, and then finally the values of the propery.

Creating in-line styles

In-line styles are CSS styles that follow an HTML element inside a .html file. It's a discouraged practice to use CSS as an in-line style as it requires an extensive amount of additional coding and is impossible to easily manipulate and reuse. It is applied similarly to how attributes are used simply by typing style="and filling in the css properties and values here".

Exploring the style element

If styles are to be coded directly into a .html file then the <style> element should be used within the head of the document. This way of coding requires a selector, propery, and value to properly display CSS. A selector is simply an HTML element or attribute that has been used on a page that we want to target with CSS styles. The properties are a family of values which all have similar characteristics, such as font, color, or border. The values are the characteristics of a property, such as bold, yellow, and sans-serif.

Basic font styling

Typography is a major aspect of content presentation so it goes without saying that a number of stylings are available to change the look of text on a page. Font-Family is a property that simply changes the fonts of text, which in itself can greatly change the mood and feel of a page on it's own. The values for font-family are typed from left to right with the default values to the right and the desired values to the left. font-weight is a property which bolds and font-style will italisize your text.

Font-size as expected adjusts the size of the text on a web page. Line-height will add or remove space between the lines of text-not the paragraphs. Text-align will shift text right, left, center, and justified. Width can be used just about anywhere and on text is often used to restrict the area that text will occupy. Another way to accomplish this is by defining the margins using the margin propery. Depending on how many values you enter or how you define the margin, you can easily control each side of the content area.

Changing color

Color may be applied to the background, forground and border of content. Once the color property has been specified you may use color values from a hexidecimal chart, RGBA, the literal name, or HSLA.

Taking styles furthur

Rather than place CSS styles in-line or within the head of an .html file we can reference an external CSS stylesheet in the head with <link rel"stylesheet" type="text/css" href="insert link here">. With this reference, now we may build a .css file which may be used by a limitless number of .html files thus greatly increasing our ability to reuse and manipulate CSS code.
