Common CSS Concepts

Working with fonts

Typography is important for design to convey what you are attempting to communicate. When styling fonts first one must consider to use System Fonts or Web Fonts. System Fonts are preinstalled fonts which come with OS software. Web Fonts utilize a font file hosted on a server and is accessed with the @font-face selector.

Formatting text

To choose a font you use the font-family property, the values of the font-family are listed from left to right with the highest priority font available listed first and the lowest priority available font listed last. Using the font-size property the size of text can be adjusted by using pixels, em's, or percetages. Em's tend to be more fluid. To specify bold text, font-weight is used while font-style creates italicized text.

Other visual elements include text-variant to create smallcaps, text-align to reposition text, text-transform to designate upper/lowercase text, text-indent, letter-spacing, word-spacing, and line-height

What is the box model?

Encapsulates all web content, consists of margins, borders, padding and the element. Used to position elements and allow them to interact with eachother. To calculate the total size of an element one must calculate the element itself along with the padding and border.

Margins and padding

Padding is the space inside elements that hold the content of an element away from its edge. Margins represent the space between elements. Margins are not factored into the total width of an element, it does need to be calculated into the total space taken up on a page. Both properties can be styled by specifying a specific size, or by using short-hand notation. Margins will collapse into each other.

Borders

Defines the outside edge of an element and is commonly used for decoration. Values can influence the width, style, and color for each individual side or through short-hand notation. The size must be added to the total element size along with padding and the content. A border-image property is currently in development to allow images to be stretched and repeated along a border. Lastly, an outline property exists which is similar to boarders but it will create an outlight around the entire element and does not add to the overall width. Since it is not widely supported in all browsers its use remains uncommon.

Backgrounds

The background extends all the way to the elements edge of the border. Properties use include background-color to change the color and apply gradients. Background-image allows the integration of an image that can take up the entire background or be repeated to create a texture. Multiple background images may be combined along with backround colors.

Color

Consists of border-color, background-color and color. Color refers to the foreground color and will effect the content of a page. Color values may be influenced using keywords, hexidecimal notation, rgba, or hsla. The a in rgba and hsla representss opacity rather than color, although it stands for alpha.

Common units of measurement

Units of measurement designated by w3c standards are <length> but when placed in application are referred to as absolute units: inches(in), centimeters(cm), millimeters(mm), points(pt), picas(pc), and relative units: ems(em), exes(ex), pixels(px), grids(gd), root ems(rem), viewport width(vw), viewport height(vh), viewport minimum(vm), and character(ch).

Viewport is a newer property which does not yet have widespread use yet has great potential for uniformity. Pixels vary depending on the screen resolution. Ems are based upon the letter 'm' of the user agent device and is great for creating a fluid layout. Relative units are recommended for web design while absolute units are generally for print.

Element positioning

Three schemes are used to place elements within a browser window. Normal Flow, float, and absolute. In Normal Flow the default, doing nothing, this stacks one element on top one another. Using the relative value of the position property you can change the position of elements without effecting it's default position, so content below will not fill the hold left behind. Using the absolute value of the position property on the other hand will allow content to be reposistioned and it's original position will be filled by with content, where this comes in handy is when an absolute value is the child of a relative value it allows you to layer content. The fixed value of the position property Allow you to scroll a window without content changing position.

Floats

Float is a property used with the values of left, right, clear or none. When an content is floated it is just like placing helium baloons on the content and having it essentially 'float' upwards on a page. Since floated content leaves gaps that other content may fill and cause unexpected resaults, you can use the clear value to make sure content still recognizes the floated content so they don't move around.

Basic layout concepts

When creating page layout, many designers will create content regions which are blocked out to allow them to visualize the look of a page prior to placing in actual content. It is recommended that percentages and ems are used to make your content accessible by all user devices.

Media types and media queries

Media types allow you to send or restrict style types based upon the device accessing the page. This is done using @media, @import, and an HTML link. You can style specifically for projectors, printers, braille, screen readers, TTY, TV, and mobile handheld devices. Media queries on the other hand are used in combination with media types to check for certain features present in the device accessing a web page to ensure that the correct layout it used when content is presented.

What is CSS3

A blanket term used to describe the functionality of CSS which has emerged over the last few years. Many new features were introduced and are still undergoing implementation. Enhancing older properties like multiple backgrounds, border-images, border-radius, and alpha values or new properties such as text-shadow, box-shadow, gradients, transitions animations, and transforms are just a few properties that make up CSS3.

Vendor prefixes

Allows browser manufacturers to support properties still in development or are proprietary to the browser. -ms-, -moz-, -o-, -khtml-, -webkit- are the prefixes used in the most popular browsers available. For developers this is a good way to see the effects of elements which may be fully implemeneted in the future and maintain wide-spread support for all users.

What is a CSS reset

This is collection of styles that nullify the default styles set by the browser. To implement, just use a wildcard selector at the top of the stylesheet and list all of the properties with their associated values that you want to become the new default. When resetting the CSS, it is important to remember to replace the CSS styles later on to the desired value. The reason to use a reset is to give a consistent baseline to build a site from.
