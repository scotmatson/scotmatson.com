Introduction To Canvas

The canvas tag is a container which allows developers to easily draw graphics onto the canvas using JavaScript. The HTML code is simple to understand but depending how the canvas is intended to be used, the JavaScript code may require a deeper understanding of advanced proramming concepts. <canvas> is a self-closing tag but to provide a fallback, web standards have dictated that alt text be placed between an opening and closing canvas tag which will display if a browser does not support this element.

Attributes

  • Width

    • Used as in-line style to determine the width of the canvas.

  • Height

    • Used as in-line style to determine the height of the canvas.

  • Global Attributes

    • Canvas supports all HTML global attributes

  • Event Attributes

    • Canvas supports all HTML event attributes
